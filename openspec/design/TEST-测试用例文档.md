| 字段 | 内容 |
|------|------|
| 接口名称 | 用户登录 |
| 请求方法 | POST |
| 请求路径 | /api/v1/auth/login |

##### 用例：正常登录

**请求参数：**
```json
{
  "username": "testuser",
  "password": "Test123456"
}
```

**预期响应：**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIs...",
    "user": {
      "id": 1,
      "username": "testuser"
    }
  }
}
```

##### 用例：密码错误

**请求参数：**
```json
{
  "username": "testuser",
  "password": "WrongPassword"
}
```

**预期响应：**
```json
{
  "code": 401,
  "message": "用户名或密码错误",
  "data": null
}
```

##### 用例：用户不存在

```json
{
  "username": "nonexist",

}
```
