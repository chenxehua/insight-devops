## 2. 概述

### 2.1 产品概述
天鹂可视化运维平台是一个面向运维团队的综合管理平台，提供部署、监控、日志分析、故障诊断等功能。平台采用响应式设计，支持桌面和移动端访问。

### 2.2 设计目标
- 简洁高效：减少操作步骤，快速达成目标
- 信息清晰：合理的信息层级和布局
- 一致性：统一的视觉语言和交互模式
- 可用性：友好的用户体验

### 2.3 技术选型
- 框架：Vue 3 + Vite
- 状态管理：Pinia
- 路由：Vue Router
- UI组件库：Ant Design Vue

### 2.4 设计规范

#### 2.3.1 设计风格
- 风格：简洁、扁平化
- 框架：基于Ant Design Pro
- 动效：轻微、适度，增强反馈感

## 6. 全局规范

### 6.1 页面跳转
- 列表页 → 详情页：跳转到详情页
- 操作成功后：留在当前页或跳转列表页（根据操作类型）
- 错误提示：Toast提示或页面内错误展示

### 6.2 消息提示

| 类型 | 使用场景 | 自动关闭 |
|------|----------|----------|
| Success | 操作成功 | 3秒 |
| Warning | 警告提示 | 5秒 |
| Error | 错误提示 | 手动关闭 |
| Info | 一般信息 | 3秒 |

### 6.3 弹窗规范

| 类型 | 使用场景 | 关闭方式 |
|------|----------|----------|
| Modal | 确认操作、表单填写 | 点击X/取消按钮/点击遮罩 |
| Drawer | 详情查看、侧边编辑 | 点击X/点击遮罩 |
| Confirm | 确认提示 | 点击确认/取消 |
