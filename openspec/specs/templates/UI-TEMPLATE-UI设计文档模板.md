### 5.2 通用组件设计

#### 5.2.1 Button 按钮

**类型：**

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| Primary | 主按钮 | 主要操作 |
| Default | 默认按钮 | 次要操作 |
| Text | 文字按钮 | 辅助操作 |
| Danger | 危险按钮 | 删除等危险操作 |

**状态：**
- 默认、悬停（Hover）、点击（Active）、禁用（Disabled）、加载中（Loading）

**尺寸：**
- 大（large）、中（middle）、小（small）

**使用示例：**
```
[ Primary ] [ Default ] [ Text ] [ Danger ]
```
